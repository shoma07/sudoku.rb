# Sudoku

Sudoku Solver

## USAGE

```
$ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/shoma07/sudoku.rb/master/sudoku.rb)" <PROBLEM FILE PATH>
```

## PROBLEM TEXT RULE

1. Blank is 0
2. Line feed every 9 columns
3. The number of lines is 9
